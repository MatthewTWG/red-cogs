# Installation
Here are the [Red](https://github.com/Cog-Creators/Red-DiscordBot) commands to add this repository. Replace `[p]` with you bot's prefix:

```text
[p]load downloader
[p]repo add red-cogs https://github.com/MatthewTWG/red-cogs
```

You may be prompted to respond with "I agree" after that.

# Cogs
You can install individual cogs with the following command. Replace `cogname` with the name of the cog you want to install.

```text
[p]cog install red-cogs cogname
```

| Cog                   | Description                                                                                                                                  | Original Author / Source |
|:----------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------|
| [misc](misc/)         | Miscellaneous server utilities with mass role management commands (`!massrole add/remove`) to add or remove roles from all server members   | MatthewTWG               |
| [streams2](streams2/) | Enhanced streams cog with platform-specific role mentions - configure different roles for Twitch, YouTube, and Picarto stream notifications  | [Red-DiscordBot](https://github.com/Cog-Creators/Red-DiscordBot) (core cog) |
| [youtube](youtube/)   | YouTube notification cog with role mentions placed before embed content instead of within the message                                        | [Mr. 42 (mr42-cogs)](https://github.com/Mister-42/mr42-cogs) |
